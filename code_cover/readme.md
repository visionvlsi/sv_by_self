#### Code cover

In the questsim

1. First compile
2. simulate
3. At vsim prompt do the following
    1. run -all
    2. coverage report -detail ( This command will display in the console itself)
    3. coverage report -detail -output c44.txt ( This command will write to the file called c44,txt)
